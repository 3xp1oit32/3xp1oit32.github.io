名词解释

  1. workspace 工作区
  2. Index / Stage 暂存区

  4. Remote （远程仓库）

新建代码库

当前目录新建一个Git代码库 

git init

新建一个目录，初始化为Git代码库 

git init [project name]

下载一个项目和它的这个那个代码历史
